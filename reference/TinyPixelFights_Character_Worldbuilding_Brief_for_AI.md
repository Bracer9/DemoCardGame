# Tiny Pixel Fights 项目与角色资料说明

本文档用于配合角色立绘，说明 **Tiny Pixel Fights** 当前已有的游戏形式、核心机制、美术风格、角色定位与技能功能。本文不包含代码实现内容。

## 1. 游戏概述

**Tiny Pixel Fights** 是一款战术 JRPG 气质的卡牌战斗游戏。当前核心是二人对战：双方各控制一支小队，通过消耗行动点进行攻击、防御、触发角色技能与状态效果，直到一方所有角色无法战斗。

当前对战模式既可以本地双人游玩，也可以在线房间对战。长期方向可能扩展为单人爬塔、英雄成长、普通兵成长、副官、遗物、天气或战场规则、剧情事件和视觉小说式演出。

游戏核心体验是小队战斗、资源取舍、角色技能互动和清楚的战斗反馈。角色不是纯数值棋子，而是带有职业身份、视觉特征、技能个性和成长潜力的卡牌单位。

## 2. 基础战斗规则

### 行动点

每回合有固定行动点，当前为 5 AP。角色攻击会消耗该角色的 Cost。当前角色 Cost 通常为 1 到 3。AP 不保留到下一回合。

### 攻击与反击

角色主动攻击敌方角色时，会同时结算攻击与反击。基础反击通常比基础攻击低 1，最低为 1。每名角色每回合最多行动一次。

### 伤害类型

当前伤害类型包括：

- **物理伤害**：武器、斧、盾、骑士、决斗等实体攻击。
- **魔法伤害**：法术、星术、自然魔法、燃烧等。
- **绝对伤害**：绕过常规防御和护盾之外部分规则的特殊伤害，直接造成 HP 损失。

### 共有盾

玩家可以消耗 AP 展开队伍共享的护盾。共有盾属于整支队伍，不属于单个角色，会先于 HP 吸收伤害。

当前共有盾规则：

- 第一次防御阵型消耗 2 AP，获得共有盾 2。
- 若当前仍有共有盾，同回合可追加 1 AP，使当前盾值 +2。
- 第二层防御会赋予队伍角色“有恃无恐”：下一次反击伤害 -1，触发后解除。
- 如果共有盾已经被打空，再次使用防御指令会重新展开第一层。

### 状态效果

当前常见状态包括：

- 治疗与祝福。
- 概率减伤与预见。
- 攻击强化。
- 燃烧等延迟伤害。
- 衰弱、驱散、攻击降低。
- 守护、代伤、共有盾。
- 特殊怒气或不可驱散强化。

所有重要状态都应在战斗中可见，并能通过预测、图标、日志、动画或音效让玩家理解发生了什么。

## 3. 美术风格

整体风格是 **暗黑中世纪战争会议桌 + 日式二次元像素 JRPG + 战术卡牌 UI**。

战场视觉是一张中世纪战争会议桌：

- 羊皮纸地图。
- 深色木桌。
- 黑铁边框。
- 旧金装饰。
- 象牙白文字或纹样。
- 深红强调色。
- 战术令牌、卡牌、盾牌和命令板。

角色立绘方向：

- 日式二次元像素 JRPG 风格。
- 参考 16/32-bit JRPG 的概括方式。
- 清楚的角色轮廓、职业符号和主色。
- 不走写实欧美奇幻厚涂，不走现代电竞或科幻 UI。
- 暗黑但不完全绝望，仍保留角色的生活感和人味。
- 每个角色应在卡牌尺寸下也能读出身份与战斗定位。

## 4. 世界观方向

当前项目的世界观方向是中世纪或近中世纪的高奇幻暗黑 JRPG。世界中可以存在王权、教会、星象、森林、边境、魔法、怪物、佣兵、普通村庄、战争遗迹和战术小队等元素。

整体气质不是纯废土，也不是明亮童话。世界经历过战争、灾难或长期衰败，但普通人仍在生活。村庄、修道院、市场、军队、佣兵、税收、医疗、传闻和日常劳动仍然存在。

当前战场表现为战争会议桌，说明这款游戏重视“指挥一支小队”的感觉。角色立绘和卡牌 UI 则强调 JRPG 式的角色存在感、成长感和技能辨识度。

## 5. 队伍与成长结构

当前长期规划中的角色结构：

- 8 名英雄。
- 每名英雄有 2 个进阶 class。
- 4 名普通兵。
- 每名普通兵有 1 个进阶 class。

英雄承担更强的角色身份、技能个性和成长路线。普通兵承担更清楚的基础职业定位，并支撑队伍构成。

## 7. 英雄角色资料

### Princess / 公主

基础定位：低 Cost 团队支援角色，攻击和生命较低，通过持续治疗维持队伍。

当前技能：**圣女的祈祷**

- 只要公主在场，己方回合开始时为全队恢复 HP。
- 治疗可以略微超过最大 HP 上限。
- 技能功能是长期续航和队伍稳定。

进阶 class：

- **High Priestess**：偏治疗、祝福、净化、庇护。
- **Dark Lord Princess**：偏命令、代价、统治、攻击强化或牺牲收益。

视觉识别点：王冠、礼服、圣性、王权、权杖、祈祷、黑红金或白金等元素。

### Oracle / 占卜师

基础定位：低 Cost 魔法支援角色，提供概率减伤和魔法增幅。

当前技能：**星读加护**

- 赋予全队预见，使受到物理或魔法伤害时有概率降低伤害。
- 赋予星读魔力，使魔法攻击和燃烧伤害提高。
- 技能功能是防御稳定和魔法队增幅。

进阶 class：

- **Astral Oracle**：偏预见、防御、星术、稳定性。
- **Fate Dealer**：偏命运、概率、标记、奖励或风险回报。

视觉识别点：星盘、玻璃球、星光、紫黑金、卡牌、骰子、命运符号。

### Peasant / 农民

基础定位：低 Cost 物理角色，拥有延迟收益和后勤节奏。

当前技能：**春播秋收**

- 在没有丰收状态时，如果作为本回合第一个进攻角色，会进行播种。
- 下个己方回合获得攻击强化。
- 技能功能是提前规划、隔回合爆发和低 Cost 行动价值。

进阶 class：

- **Harvest Guard**：偏民兵、防线、粮道、护盾或团队保护。
- **Harvest Saint**：偏丰收、治疗、祝福、生命恢复或资源收益。

视觉识别点：麦穗、篮子、农具、草叉、泥土、绿、棕、奶油色、朴素布料。

### Mage / 魔法师

基础定位：中 Cost 高攻击魔法输出角色，擅长延迟魔法伤害。

当前技能：**灼热刻印**

- 攻击时有概率赋予敌人燃烧。
- 燃烧会在目标下回合开始时造成魔法伤害。
- 星读魔力可以提高燃烧伤害。
- 技能功能是魔法压力、延迟伤害和状态输出。

进阶 class：

- **Stellar Archmage**：偏高阶星术、魔法爆发、蓄力输出。
- **Arcane Archivist**：偏书卷、记录、复制、封印或知识管理。

视觉识别点：法杖、火焰、红晶、黑袍、书、符文、星火、魔法阵。

### Knight / 骑士

基础定位：高 Cost 高 HP 防御角色，承担保护队友和吸收伤害的职责。

当前技能：**替身之盾**

- 每回合替骑士以外的队友承受一次主动物理伤害。
- 对反击、魔法和部分技能无效。
- 技能功能是保护、代伤和维持队伍防线。

进阶 class：

- **Dread Cavalier**：偏冲锋、压迫、破阵、攻击性防御。
- **Holy Paladin**：偏圣盾、庇护、净化、稳定防线。

视觉识别点：盾、长枪、剑、盔甲、战马、誓约、红银、白金或黑甲。

### Druid / 德鲁伊

基础定位：低 Cost 魔法控制角色，擅长衰弱和驱散敌方强化。

当前技能：**衰弱孢子**

- 攻击造成伤害时稳定发动，未造成伤害时有概率发动。
- 发动后移除目标一个可驱散的攻击强化。
- 赋予目标待生效衰弱，使其下回合进攻伤害降低。
- 技能功能是削弱输出、反制强化和控制节奏。

进阶 class：

- **Grove Keeper**：偏恢复、净化、森林守护、生命循环。
- **Wildspeaker**：偏野性、猎群、自然意志、攻击性控制。

视觉识别点：藤蔓、叶片、蘑菇、孢子、鸟、鹿角、野花、深绿、褐色。

### Barbarian / 狂战士

基础定位：中 Cost 高攻击物理输出角色，擅长通过重击制造波及伤害。

当前技能：**战斧余波**

- 主动攻击对目标造成足够伤害时，对相邻敌人造成额外物理伤害。
- 技能功能是破阵、压血和多目标压力。

进阶 class：

- **Radiant Berserker**：偏光辉狂战、荣耀、爆发、强化攻击。
- **Dragon Raider**：偏龙袭、破盾、机动、掠夺式进攻。

视觉识别点：巨斧、兽皮、尖刺、红黑、边境、伤疤、龙、断旗。

### Monster / 怪物

基础定位：高 Cost 高 HP 特殊攻击角色，擅长反防御和绝对伤害。

当前技能：**美女与野兽**

- 对公主以外的角色主动攻击未造成伤害时，追加绝对伤害。
- 如果我方公主存活，追击伤害提高。
- 任意公主被击败后，怪物获得野兽之怒，基础攻击提高。
- 技能功能是反制减伤、防御和特殊关系触发。

进阶 class：

- **Mirror Fiend**：偏镜像、反射、复制、身份错位。
- **Abyssal Queen**：偏深渊、支配、绝对伤害、怪物王权。

视觉识别点：角、尾、爪、破斗篷、黑红、深渊、镜子、少女感与兽性并存。

## 9. 普通兵资料

### Cleric / 牧师

基础定位：低 Cost 魔法普通兵，偏治疗、净化和基础支援。

队伍作用：

- 与治疗、净化和承受代价的队伍方向相容。
- 可以帮助维持低 HP 角色或高风险角色。

进阶 class：

- **Saint Cleric**：更强治疗、净化、祝福或持续支援。

视觉识别点：白布、圣铃、绷带、念珠、蜡、圣水、治疗道具。

### Shieldmaiden / 盾女

基础定位：中 Cost 物理普通兵，低攻击、高防御，偏共有盾和保护。

队伍作用：

- 与盾、防御、守护和蓄力队伍方向相容。
- 可以强化队伍共同防线。

进阶 class：

- **Aegis Shieldmaiden**：更强共有盾、防御、魔法减伤或团队保护。

视觉识别点：大盾、修补痕迹、铁、皮革、护送队、朴素纹章。

### Duelist / 决斗者

基础定位：低 Cost 高攻击物理普通兵，生命较低，适合攻击已被标记或带有负面状态的目标。

队伍作用：

- 与标记、衰弱、燃烧、猎物等状态配合。
- 适合收割、斩杀和低 Cost 进攻节奏。

进阶 class：

- **Crimson Duelist**：更强执行、收割、状态目标打击或连击。

视觉识别点：细剑、红衣、手套、轻甲、决斗姿态、精准攻击。

### Arcanist / 奥术师

基础定位：中 Cost 魔法普通兵，偏魔法增幅、状态放大和魔法辅助。

队伍作用：

- 与魔法伤害、燃烧、星术、衰弱和魔法标记配合。
- 可以强化魔法输出或降低敌方魔法防御。

进阶 class：

- **Astral Arcanist**：更强星术聚焦、魔法增幅、魔防削弱或状态放大。

视觉识别点：仪器、星盘、工具包、羊皮纸、公式、低阶法杖、学徒长袍。
