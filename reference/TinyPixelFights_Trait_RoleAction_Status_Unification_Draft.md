# Tiny Pixel Fights — Trait / Role Action 状态整合实装基准

## 1. 整合原则

### 1.1 通用状态优先

以后新增 Trait / Role Action 时，优先复用下方通用状态。没有写在本表里的效果，不要硬塞进通用状态。

例如物防/魔防直接提高、每回合回血、概率发动、特殊印记、身份变化、资源变化等，默认都是特殊状态或系统效果。

### 1.2 回合型和次数型必须分开命名

同样是“攻击变强”，如果一个持续若干 turn，另一个只作用于下一次攻击，它们就不是同一个状态。

- 回合型物理强化叫“强攻”。
- 下一次物理强化叫“猛击”。
- 回合型魔法强化叫“魔涌”。
- 下一次魔法强化叫“咏唱”。

这不是文字洁癖，是玩家认知边界。否则 UI 合并后玩家会不知道这个状态到底是“这回合有效”还是“下一次有效”。

### 1.3 叠加规则

- “回合数可叠加，效果不叠加”：重复获得时延长持续 turn，倍率仍只算一次。
- “次数可叠加，效果不叠加”：重复获得时增加次数，每次触发只按一次倍率计算。
- 层数型状态按层数消耗，例如燃烧。
- 战栗只能一次性赋予，持续 turn 不叠加。

这里的“回合”指一方玩家的一次行动 turn，不是 round。

### 1.4 默认可驱散

本表中的通用 Buff / Debuff 默认可以被驱散或净化。游戏文本里不需要每次写“可驱散”；只有不可驱散才需要特别写。

光环类效果不是普通 Buff，不进入驱散池。

## 2. 通用状态规格

| 通用状态 | 类型 | 作用轴 | 规则 |
| --- | --- | --- | --- |
| 强攻 | Buff | 物理输出 / 普通攻击 | 按 turn 生效。主动物理 / 普通攻击伤害 ×1.5，向上取整。持续 turn 可叠加，效果不叠加。 |
| 猛击 | Buff | 物理输出 / 普通攻击 | 下一次主动物理 / 普通攻击伤害 ×2。次数可叠加，效果不叠加。 |
| 魔涌 | Buff | 魔法输出 / 普通攻击 | 按 turn 生效。魔法伤害 / 普通攻击伤害 ×1.5，向上取整。持续 turn 可叠加，效果不叠加。 |
| 咏唱 | Buff | 魔法输出 | 下一次魔法伤害 ×2。次数可叠加，效果不叠加。 |
| 坚守 | Buff | 物理承伤 | 按 turn 生效。受到物理伤害 ×0.5。持续 turn 可叠加，效果不叠加。 |
| 护咒 | Buff | 魔法承伤 | 按 turn 生效。受到魔法伤害 ×0.5。持续 turn 可叠加，效果不叠加。 |
| 脆弱 | Debuff | 物理承伤 | 按 turn 生效。受到物理伤害 ×1.25，向上取整。持续 turn 可叠加，效果不叠加。 |
| 空虚 | Debuff | 魔法承伤 | 按 turn 生效。受到魔法伤害 ×1.25，向上取整。持续 turn 可叠加，效果不叠加。 |
| 力竭 | Debuff | 物理输出 | 按 turn 生效。造成物理伤害 ×0.5，向下取整。持续 turn 可叠加，效果不叠加。 |
| 磨损 | Debuff | 魔法输出 | 按 turn 生效。造成魔法伤害 ×0.5，向下取整。持续 turn 可叠加，效果不叠加。 |
| 燃烧 / 炎上 | Debuff | 持续伤害 | 己方 turn 开始时消耗 1 层并受到 1 点魔法伤害。可叠层，伤害不叠加；后端保留伤害可被来源或光环提高的接口。 |
| 战栗 | Debuff | 反击限制 | 按 turn 赋予。无法反击。只能一次性赋予，持续 turn 不叠加。 |

## 3. 特殊状态边界

以下不强行归入通用状态：

- 公主的治疗、净化、祝福类效果。
- 怪物的野兽之怒、美女与野兽、黑暗契约。
- 农民的播种 / 丰收循环。
- 共享盾、AP、BP、奖励成长。
- 概率发动的印记、预见、星读加护、守护等 Aura。
- 直接提高 / 降低物防或魔防的数值状态，除非明确改成坚守 / 护咒 / 脆弱 / 空虚。

| --- | --- | --- | --- |
| 公主 | `saints-prayer` | 维持现有治疗光环。 | Aura / 特殊治疗 |
| 占卜师 | `stargazers-aegis` | 受到伤害时，30% 使该伤害 ×0.5。我方魔法和燃烧伤害 +1。 | Aura，不作为普通 Buff |
| 农民 | `spring-harvest` | 可播种的 turn 内，只要农民是第一个发起主动攻击的角色即可触发；Role Action 不会破坏这个条件。 | 特殊循环 |
| 魔法师 | `searing-mark` | 主动攻击结算后，若目标仍存活，50% 赋予 1 层燃烧。 | 燃烧 |
| 德鲁伊 | `weakening-spores` | 触发概率前提保持不变。发动后随机移除目标 1 个可驱散 Buff，并赋予目标 2 turn 力竭和磨损；即刻生效。 | 力竭 + 磨损 |

| 怪物 | `predatory-instinct` | 对非公主目标主动攻击造成 0 点 HP 伤害时，追击当前怪物攻击力点绝对伤害；若我方公主存活，追击 +1。攻击公主时造成当前怪物攻击力点绝对伤害，攻击后怪物受到造成伤害 2 倍的自伤，但不会因此死亡。任意公主死亡后，怪物获得野兽之怒：基础攻击 +2。 | 特殊 Trait / 野兽之怒 |
| 骑士 | `interposing-shield` | 对骑士以外我方角色受到的主动物理攻击，全队共享 1 次：我方将受到 1 点及以上主动物理伤害时，骑士承担目标受到伤害的 1/3，向上取整；该承担伤害会再受骑士自身物防影响。 | Aura / 反应规则 |

## 5. Role Action 调整基准

| 角色 | Role Action ID | 调整后规则 | 状态归类 |
| --- | --- | --- | --- |
| 公主 | `saintly-prayer` | 治疗目标 2 HP；若目标有可净化 Debuff，净化 1 个并获得 1 AP。 | 治疗 / 净化 / AP |
| 公主 | `royal-command` | 维持现有 AP 经济与 CD。 | 资源规则 |
| 骑士 | `guard-oath` | 自身以外的我方角色获得 1 层守护誓约。受到主动物理攻击时，伤害 -2 并消耗 1 层。可叠层，可驱散；驱散时整类移除。 | 特殊次数 Buff，不归入通用坚守 |
| 骑士 | `raise-bulwark` | 我方拥有共享盾时可发动；共享盾层数 ×1.5，向上取整。共享盾物防 +2，持续 2 turn。 | 盾规则 / 特殊物防 |
| 狂战士 | `war-cry` | 自身获得狂怒：物防 / 魔防 -2，持续到下个己方 turn；本 turn 主动攻击次数 +1；若击破共有盾，对攻击对象追加当前攻击力点绝对伤害。 | 特殊强化 / 行动次数 |
| 狂战士 | `challenge` | 目标获得战栗，1 turn 不能反击。 | 战栗 |
| 魔法师 | `arcane-channel` | 本 turn 不能再主动攻击；下个己方 turn 获得 2 层咏唱。 | 咏唱 |
| 魔法师 | `searing-brand` | 目标获得 1 层燃烧，并获得 1 层空虚。 | 燃烧 + 空虚 |
| 占卜师 | `star-reading` | 维持现有再攻击规则。 | 行动次数系统 |
| 占卜师 | `fate-mark` | 维持命运标记。 | 特殊印记 |
| 怪物 | `predatory-gaze` | 目标获得猎物：本 turn 每次受到 0 点 HP 伤害时，追加 2 点绝对伤害。 | 特殊印记 |
| 怪物 | `dark-pact` | 保留黑暗契约为特殊状态，不归入强攻或猛击。 | 特殊 Buff |
| 农民 | `supply-basket` | 治疗我方 1 体 1 HP，并赋予 2 turn 坚守。 | 治疗 + 坚守 |
| 农民 | `field-work` | 维持农民播种 / 丰收分支；播种中时改为回复自己 2 HP。 | 特殊循环 |
| 德鲁伊 | `cleansing-herbs` | 维持净化草药。 | 净化 / 治疗 |
| 德鲁伊 | `weakening-spores-action` | 随机移除目标 1 个可驱散 Buff，并施加 2 turn 力竭和磨损。 | 驱散 + 力竭 + 磨损 |

## 6. 来源 → 状态映射

| 来源 | 迁移目标 | 持续方式 |
| --- | --- | --- |
| 魔法师 Trait `searing-mark` | 燃烧 | 层数 |
| 德鲁伊 Trait `weakening-spores` | 力竭 + 磨损 | 2 turn |
| 骑士 Role Action `guard-oath` | 守护誓约 | 次数型；每层抵挡 1 次主动物理攻击 |
| 狂战士 Role Action `challenge` | 战栗 | 1 turn，不叠加 |
| 魔法师 Role Action `arcane-channel` | 咏唱 x2 | 下个己方 turn 获得，次数型 |
| 魔法师 Role Action `searing-brand` | 燃烧 x1 + 空虚 x1 | 燃烧层数；空虚按 turn |
| 农民 Role Action `supply-basket` | 坚守 | 2 turn |
| 德鲁伊 Role Action `weakening-spores-action` | 力竭 + 磨损 | 2 turn |

## 7. 实装注意

1. 通用状态可以共用显示图标和一级 UI 名称，但后端仍要保留具体来源 instance，方便驱散随机移除一个来源。
2. 同名回合型状态延长 turn，不叠倍率。
3. 同名次数型状态增加次数，不叠倍率。
4. Aura 不作为 Buff，不可驱散。
5. 游戏文本里不要写“可驱散”；只有不可驱散才写。
